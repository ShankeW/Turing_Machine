# Project Description
This small project is a Turing machine simulator. A Gödel number can be entered and analyzed into a Turing machine.

The analyzed Turing machine can start a computation using an input. The step mode allows every step of the computation to be observed.

Visible at each computation step are:
 - The machine head
 - The tape
 - The current state
 - The number of computation steps

# User Instructions
1. To enter the desired Gödel number, go to `src/main/resources/goedelnumber.txt`. Press Enter to start a new line and enter the Gödel number.
2. Run the main class.
3. Select the line number where the Gödel number is located in the `goedelnumber.txt` file.
4. Select a mode:
   - Run mode provides the result directly.
   - Step mode displays every computation step. In step mode, you can also set the execution delay for each step.

# Projektbeschreibung
Dieses kleine Projekt ist ein Turingmaschinen-Simulator. Eine Gödelnummer kann eingegeben und zu einer Turingmaschine analysiert werden.

Die analysierte Turingmaschine kann mit einer Eingabe eine Berechnung starten. Mit dem Step-Modus kann jeder Berechnungsschritt beobachtet werden.

Ersichtlich bei jedem Berechnungsschritt sind:
 -  Der Maschinenkopf
 -  Das Band
 -  Der Aktuelle Zustand
 -  Die Anzahl Berechnungsschritte
  
# Bedienungsanleitung
1. Um die gewünschte Gödelnummer einzugeben, gehe zu `src/main/resources/goedelnumber.txt`. Starte mit Enter eine neue Zeile und gib die Gödelnummer ein.
2. Starte die Main-Klasse.
3. Wähle nun die Zeilennummer aus, auf der sich die Gödelnummer in der Datei `goedelnumber.txt` befindet.
4. Wähle einen Modus aus:
   - Der Run-Modus liefert die Antwort direkt.
   - Der Step-Modus zeigt jeden Berechnungsschritt an. Für den Step-Modus kann man zusätzlich noch die Zeit für jeden Berechnungsschritt bestimmen.
